### Description

**NAND**

- Add your themes to `Theme_randomizR.hmod\usr\share\themes`.
- Install Hmod

- Install Hmod
- Add your themes folders to `MEDIA:\hakchi\themes`.

**MAX 60 Themes** more will lead to booting issues + **C8**
